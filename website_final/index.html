Applied Deep Learning: A Deployed Keras Model for Columbia Landmarks

This applied example is based on the Applied Deep Learning (COMS 4995) course at Columbia University and this github repo.

This website uses a convolutional neural network (CNN) with roughly 1,000,000 parameters of size (~10MB) to recognize landmarks (Scholar's Lion, Alma Mater, & Le Penseur) on the Columbia University campus in NYC.

This lightweight model was trained on a data-set consisting of 150 images.
